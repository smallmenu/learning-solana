# spl-token

Token-2022 允许铸币厂创建者包含 他们的代币元数据直接存储在 Mint 账户中

Token Program 2022: TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb


## 常用指令

```bash
# 查看所有的代币
spl-token accounts

# 查看代币余额（代币地址）
spl-token balance 2TPEdMFLNnjw7QyjU3H4VzJzgMMyjRpWzgmar16pg7Cs

# 查看代币信息（代币地址）
spl-token display 2TPEdMFLNnjw7QyjU3H4VzJzgMMyjRpWzgmar16pg7Cs
```


## Token

### 创建普通代币

```bash
spl-token create-token --decimals 6
```

提示如下，创建代币地址是：`D5XMvoQy7hgrTh1q6Nsoh39Qbrn7MAPtbabNeLZ9bgMG`：

```bash
Creating token D5XMvoQy7hgrTh1q6Nsoh39Qbrn7MAPtbabNeLZ9bgMG under program TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA

Address:  D5XMvoQy7hgrTh1q6Nsoh39Qbrn7MAPtbabNeLZ9bgMG
Decimals:  6
```

### 创建代币帐户

```bash
spl-token create-account D5XMvoQy7hgrTh1q6Nsoh39Qbrn7MAPtbabNeLZ9bgMG
```

提示如下：

```bash
Creating account 95zT2dghmmNnVLRgEQ1eGCEWtmVGvSu67earsotZ6w2M
```

### 铸造代币

```bash
spl-token mint D5XMvoQy7hgrTh1q6Nsoh39Qbrn7MAPtbabNeLZ9bgMG 10000
```

代币会铸造到代币帐户中，即：`95zT2dghmmNnVLRgEQ1eGCEWtmVGvSu67earsotZ6w2M`

可通过以下方式查看：

```bash
spl-token accounts
spl-token balance D5XMvoQy7hgrTh1q6Nsoh39Qbrn7MAPtbabNeLZ9bgMG
```
