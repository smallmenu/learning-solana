# solana 常用命令

## 基础配置

solana CLI 默认钱包位置：`~/.config/solana/id.json`，里面是密钥

```bash
# 查看当前配置
solana config get

# 设置 RPC 切换环境
solana config set --url mainnet-beta
solana config set --url devnet
solana config set --url localhost

# 可以指定 RPC
solana config set --url https://devnet-rpc.shyft.to?api_key=m4T8LXSoiymOiiHM
```

## 常用指令

```bash
# 查看当前默认钱包地址
solana address

# 空投（如在 devnet）
solana airdrop 5

# 查看余额
solana balance
```
